# App_19_ResponsiveAdaptive

## Aplicació Responsive i Adaptive
- Exemple d'aplicació responsive i adaptive
- Té 3 layouts possibles
  - MyBasicLayout: un layout on només mostra el tamany de cada coordenada i l'orientació
  - ChessGameScreen: Un joc d'escacs amb diferent layout segons sigui PORTRAIT o LANDSCAPE
  - AdaptiveChessScreen: (fet a MainActivityAdvanced) El mateix joc d'escacs, però amb una gestió diferent de l'orientació
